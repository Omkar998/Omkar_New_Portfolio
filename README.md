# Omkar Portfolio (Static)

## Run locally
Just open `index.html` in a browser, or use a tiny server:

```bash
python -m http.server 8000
```

Then open: http://localhost:8000

## Customize
- Replace the placeholder LinkedIn/GitHub links in `index.html`.
- Update project links (Live/Code/Report).
- If you want to add a profile photo, drop it in this folder and add an `<img>` in the hero.

## Deploy to GitHub Pages
1. Create a repo (e.g., `omkar-portfolio`)
2. Upload these files to the repo root
3. Settings → Pages → Deploy from branch → `main` / root
